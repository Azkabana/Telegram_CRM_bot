# Telegram CRM с ИИ-помощником

Telegram-CRM для обработки заявок с хранением в PostgreSQL
и ИИ-помощником для генерации ответов клиентам.

## Стек
- Python 3.13
- aiogram 3
- PostgreSQL
- OpenAI API

## Функции
- Приём и публикация заявок
- Статусы заявок
- Поиск по заявкам
- Генерация ответов через ИИ
- Хранение данных в PostgreSQL

## Запуск проекта
1. Склонировать репозиторий
2. Создать .env на основе .env.example
3. Установить зависимости: pip install -r requirements.txt
4. Запустить бота
